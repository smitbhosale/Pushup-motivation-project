The Pushup Motivation Project is a simple and interactive application designed to motivate users to complete daily pushups by tracking repetitions, providing encouraging messages, and helping build consistency in fitness routines. This project promotes healthy habits using a fun and engaging approach.

⚙️ Features

✅ Real-time pushup counter

✅ Motivational quotes/messages display

✅ Session progress tracking

✅ Simple and user-friendly interface

✅ Lightweight and fast execution




How to Run the Project:


Clone the repository:

git clone <your-repository-link>



Navigate to the project folder:

cd pushup-motivation-project



Run the program:

python main.py
